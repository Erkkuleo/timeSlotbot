class CalendarEvent(var startTime: String, var endTime: String)

Represents a single event with a starting time and an ending time.

Value parameters:
endTime

The ending time as a String. Is in format yyyyMMddThhmmssZ (this is due to the formatting in the .ics-files)

startTime

The starting time as a String. Is in format yyyyMMddThhmmssZ (this is due to the formatting in the .ics-files)
